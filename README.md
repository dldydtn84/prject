# 애플유저사용자 커뮤니티

--------------------------------------

## 개요
- 애플제품을 사용하는 사용자들의 실시간 소통이 가능하며 각 게시판을 이용하여 정보공유및 자료공유가 가능하다.

## 개발환경
- IDE : IntelliJ 2019.02
- Language : Java 11
- Framework : Spring Boot, Spring Security
- DB : MySQL
- UI : Bootstrap
- ETC : Mybatis, lombok
- Editor : Summernote

## 배포환경
 - Server : AWS EC2 centOS
 - DB : AWS RDS MySQL
